# Ecommerce-sales-analysis
Project Overview
E-commerce sales analysis project using python and power BI with interactive dashboard and business insights.
## Tools used
- Python (Pandas, Numpy)
- SQLite (SQL Queries)
- Power BI
- MS Excel
-------
## Key Analysis performed 
- Data cleaning & preprocessing
- sales and profit analysis
- Top performing products
- Region-wise performance
- Profit vs Loss Distribution
-------
Key Insights
- West region generates the highest sales
- Technology category is the most profitable
- Top 10 products contribute major sales
- Around 19% transactions are in loss
- Sales show a stedy growth trend
